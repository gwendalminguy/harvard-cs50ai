# Language

This directory focuses on the seventh lecture of the class about language, covering the following subjects:

- Syntax
- Semantics
- Context-Free Grammar
- `nltk`
- n-grams
- Bag-of-Words Model
- Naive Bayes
- Word Representation
- `word2vec`
- Attention
- Transformers

It contains two projects:

* [parser](https://github.com/gwendalminguy/harvard-cs50ai/tree/main/language/parser), a program that parses sentences and extract noun phrases.
* [attention](https://github.com/gwendalminguy/harvard-cs50ai/tree/main/language/attention), a program that predicts a masked word in a text sequence and generates attention diagrams.
